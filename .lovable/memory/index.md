# Project Memory

## Core
Ensight consulting site. Deep teal-slate (#0E2D3D) & ocean teal (#0A7EA4) palette. Warm amber (#C8973F) for stat highlights. Dark section (#0B1E27) for quote block. No monospace labels. font-semibold for h2, font-bold for h1. Sharp corners (0.5rem radius). No `//` prefix on section labels. Thin (2px) card accent bars. Single-color nav bar. Plus Jakarta Sans 800 headings, Outfit body.
