# Project Memory

## Core
Ensight consulting site. Deep teal-slate & electric teal palette. Warm amber (#E8A838) for stat highlights. No monospace labels. font-semibold for h2, font-bold for h1. Sharp corners (0.5rem radius). No `//` prefix on section labels. Thin (2px) card accent bars. Single-color nav bar, not gradient rainbow. Plus Jakarta Sans 800 for h1/h2 headings, Outfit for body text.
